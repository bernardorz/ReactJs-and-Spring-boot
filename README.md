<h2>🔌 Getting started</h2>
Clone the project:

```bash
$ git clone https://github.com/Goytai/Podcastr.git
```

Acess the folder:

```bash
$ cd Podcastr
```

Follow the steps below:
```bash
# Install the dependencies
$ yarn

# Start the project
$ cd ./frontend 
$ yarn start
```
